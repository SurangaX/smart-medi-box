## вҸұпёҸ Token Lifecycle

```
SESSION TOKEN (7 days)
в”ңв”Җв”Җ Created: User signup or login
в”ңв”Җв”Җ Stored: session_tokens table
в”ңв”Җв”Җ Used: Every authenticated API call
в”ңв”Җв”Җ Expires: 7 days after creation
в””в”Җв”Җ Action: User must login again

PAIRING TOKEN (1 hour)
в”ңв”Җв”Җ Created: Patient requests device pairing
в”ңв”Җв”Җ Stored: pairing_tokens table
в”ңв”Җв”Җ Used: Device sends with MAC address
в”ңв”Җв”Җ Expires: 1 hour after creation
в”ңв”Җв”Җ Reusable: No (marked is_used = true)
в””в”Җв”Җ Action: Generate new token if expired
```
